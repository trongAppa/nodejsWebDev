파일 읽기와 쓰기
파일을 읽고 쓰는 작업은 프로그래밍에서 자주 사용되는 기능 중 하나
Node.js에서는 fs 모듈을 사용하여 파일을 비동기적으로 읽고 쓸 수 있음

1) 파일 읽기
파일을 읽을 때는 fs.readFile 함수를 사용
함수는 비동기적으로 파일 내용을 읽어와서 콜백 함수를 호출

fs.readFile 함수를 사용하여 파일을 읽는 예제

const fs = require('fs');

fs.readFile('파일경로', '인코딩', (err, data) => {
  if (err) {
    console.error('파일 읽기 실패:', err);
    return;
  }

  console.log('파일 내용:', data);
});

'파일경로'는 읽을 파일의 경로를 나타내며, '인코딩'은 파일의 인코딩 방식을 나타남
파일을 성공적으로 읽으면 콜백 함수가 호출되어 읽은 파일 내용이 data 매개변수로 전달

2) 파일 쓰기
파일을 쓸 때는 fs.writeFile 함수를 사용
비동기적으로 파일에 내용을 작성

fs.writeFile 함수를 사용하여 파일을 쓰는 예제

const fs = require('fs');

const content = '파일에 쓸 내용';

fs.writeFile('파일경로', content, '인코딩', (err) => {
  if (err) {
    console.error('파일 쓰기 실패:', err);
    return;
  }

  console.log('파일 쓰기 완료');
});

'파일경로'는 쓸 파일의 경로를 나타내며, content는 파일에 쓸 내용
파일을 성공적으로 쓰면 콜백 함수가 호출

3) 파일 읽기와 쓰기 예외 처리
파일 읽기와 쓰기 작업은 실패할 수 있으므로 예외 처리는 필요
콜백 함수의 첫 번째 매개변수 err을 확인하여 작업 실패 시 적절한 처리를 할 수 있음
