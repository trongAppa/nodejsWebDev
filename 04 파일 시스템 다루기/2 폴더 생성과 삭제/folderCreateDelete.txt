폴더 생성과 삭제
프로그램에서 폴더를 생성하고 삭제하는 작업은 때로는 필요할 수 있음
Node.js에서는 fs 모듈을 사용하여 폴더를 생성하고 삭제할 수 있음

1) 폴더 생성
폴더를 생성할 때는 fs.mkdir 함수를 사용
비동기적으로 폴더를 생성

fs.mkdir 함수를 사용하여 폴더를 생성하는 예제

const fs = require('fs');

fs.mkdir('폴더경로', (err) => {
  if (err) {
    console.error('폴더 생성 실패:', err);
    return;
  }

  console.log('폴더 생성 완료');
});

'폴더경로'는 생성할 폴더의 경로를 나타냄 / 폴더 생성이 성공하면 콜백 함수가 호출

2) 폴더 삭제
폴더를 삭제할 때는 fs.rmdir 함수를 사용
비동기적으로 폴더를 삭제

fs.rmdir 함수를 사용하여 폴더를 삭제하는 예제

const fs = require('fs');

fs.rmdir('폴더경로', (err) => {
  if (err) {
    console.error('폴더 삭제 실패:', err);
    return;
  }

  console.log('폴더 삭제 완료');
});

'폴더경로'는 삭제할 폴더의 경로를 나타냄 / 폴더 삭제가 성공하면 콜백 함수가 호출

3) 폴더 생성과 삭제 예외 처리
폴더 생성과 삭제 작업은 실패할 수 있으므로 예외 처리는 필요
콜백 함수의 첫 번째 매개변수 err을 확인하여 작업 실패 시 적절한 처리 가능
