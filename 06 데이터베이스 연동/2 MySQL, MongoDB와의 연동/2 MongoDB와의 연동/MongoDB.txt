MongoDB와의 연동
MongoDB 모듈 설치
- npm install mongodb

연결 설정
- MongoDB 데이터베이스에 연결하기 위해 필요한 호스트, 포트, 데이터베이스 이름 등의 정보를 설정

const MongoClient = require('mongodb').MongoClient;

const url = 'mongodb://localhost:27017';
const databaseName = 'database_name';

MongoClient.connect(url, (error, client) => {
  if (error) throw error;

  const db = client.db(databaseName);

  // 쿼리 실행
});

쿼리 실행: 연결 설정이 완료되면 db.collection() 메서드를 사용하여 컬렉션에 대한 쿼리를 실행

const collection = db.collection('collection_name');

collection.find({}).toArray((error, documents) => {
  if (error) throw error;

  // 결과 처리
});

Node.js에서 MySQL과 MongoDB와의 연동을 위해 필요한 모듈 설치, 연결 설정, 쿼리 실행 방법을 간단히 소개
원하는 데이터베이스와의 연동을 수행하고, 데이터의 삽입, 조회, 수정, 삭제 등을 수행
데이터베이스 연동은 Node.js 애플리케이션에서 중요한 부분이므로, 데이터베이스의 특성과 요구사항에 맞게 적절한 방법을 선택하여 사용
