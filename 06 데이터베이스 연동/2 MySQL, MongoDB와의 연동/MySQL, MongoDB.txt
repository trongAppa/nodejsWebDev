MySQL, MongoDB와의 연동
MySQL
- 관계형 데이터베이스 관리 시스템(RDBMS)
- 데이터를 테이블 형태로 구조화하여 저장하고 관리
- 다양한 응용 프로그램과 웹 애플리케이션에서 널리 사용되는 대표적인 RDBMS
- 표준 SQL 문법을 사용하여 데이터를 조작하고 질의할 수 있으며, ACID(Atomicity, Consistency, Isolation, Durability) 특성을 지원하여 데이터의 일관성과 안정성을 보장
- 대규모 데이터베이스 시스템을 다루는 데 효율적이며, 많은 기능과 확장성을 제공

MongoDB
- 비관계형 데이터베이스 관리 시스템(NoSQL DBMS)
- 문서 지향적인 데이터 저장 방식을 제공
- 데이터는 JSON 형태의 BSON(Binary JSON) 문서로 저장되며, 컬렉션(Collection)과 문서(Document)의 구조로 구성
- 유연한 스키마를 가지고 있어 데이터 구조를 동적으로 변경할 수 있으며, 확장성이 뛰어나 분산 데이터베이스 환경에서도 우수한 성능을 발휘
- 쉽게 확장 가능한 클러스터링, 샤딩(Sharding), 복제(Replication) 등의 기능을 제공하여 대용량의 데이터 처리에 유용
- 자바스크립트 기반의 쿼리 언어를 사용하여 데이터를 조작하고 검색

MySQL : 관계형 데이터베이스로 구조화된 데이터를 저장하고 다루는 데에 특화
MongoDB : 비관계형 데이터베이스로 유연한 데이터 구조와 확장성을 갖추고 있음
