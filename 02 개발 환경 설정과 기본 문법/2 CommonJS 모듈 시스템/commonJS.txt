CommonJS 모듈 시스템
- Node.js에서 모듈을 효율적으로 정의하고 사용하기 위한 방법을 제공
- 코드의 재사용성을 높이고, 모듈 간의 의존성을 관리
- 모듈 시스템은 Node.js 생태계에서 널리 사용되며, Node.js 개발에서 중요한 개념 중 하나

1) 모듈의 정의
모듈은 Node.js에서 파일 단위로 정의되며, 각 파일은 개별적인 모듈로 취급
코드를 모듈화하여 재사용 가능하고 관리하기 쉬운 구조로 구성
모듈의 정의는 module.exports 객체를 사용하여 수행
module.exports 객체는 해당 모듈에서 외부로 내보낼 함수, 객체, 변수 등을 정의하는 데 사용
다른 파일에서 해당 모듈을 가져와 사용할 수 있음

모듈의 정의 방법
1. 함수 내보내기

function greet(name) {
   console.log(`Hello, ${name}!`);
}

module.exports = greet;

2. 객체 내보내기
const person = {
   name: "John",
   age: 30,
};

module.exports = person;

모듈의 정의를 통해 코드를 모듈화하고 외부로 내보내어 다른 파일에서 사용할 수 있도록 만들 수 있음
코드의 재사용성을 높이고, 모듈 간의 의존성을 관리가능

2) 모듈의 가져오기
다른 모듈을 현재 모듈로 가져오는 것을 의미
다른 모듈에서 정의한 함수, 객체, 변수 등을 현재 모듈에서 사용할 수 있음
require() 함수를 사용하여 수행

1. require() 함수 사용
const greet = require("./greet");

require() 함수의 인자로 가져오고자 하는 모듈의 파일 경로를 전달
(파일 경로는 상대 경로 또는 절대 경로를 사용할 수 있음)

3) 모듈의 사용
가져온 모듈의 함수, 객체, 변수 등을 현재 모듈에서 활용하는 것을 의미
모듈을 가져온 후 해당 모듈의 내보낸 항목을 참조하여 사용

모듈의 사용 방법
1. 가져온 모듈의 함수, 객체, 변수 등을 사용
const greet = require("./greet");

greet("AI");

greet 모듈을 가져온 후, greet 함수를 호출하여 "AI"이라는 인자를 전달 / 모듈이 수행하는 기능을 활용

2. 모듈의 내보낸 항목을 변수에 할당하여 사용
const person = require("./person");

console.log(person.name); // "John"
console.log(person.age); // 30

person 모듈을 가져온 후, person 객체의 속성을 참조하여 값을 출력 / 모듈이 내보낸 객체의 데이터를 사용

가져온 모듈의 함수, 객체, 변수 등을 활용하여 원하는 기능을 수행
모듈 간의 상호작용을 구현하고 코드를 모듈화하여 재사용성을 높일 수 있음

4) 모듈의 내보내기
현재 모듈에서 외부로 내보낼 함수, 객체, 변수 등을 정의하는 것을 의미
다른 파일에서 해당 모듈을 가져와 사용할 수 있음
module.exports 객체에 값을 할당하여 수행

모듈의 내보내기 방법
1. module.exports 객체에 값을 할당하여 해당 모듈에서 외부로 내보낼 항목을 정의
function greet(name) {
   console.log(`Hello, ${name}!`);
}

module.exports = greet;

greet 함수를 module.exports에 할당하여 외부로 내보내고 있음 / 다른 모듈에서 require() 함수로 해당 모듈을 가져와 사용할 수 있음

2. 할당하는 값은 함수, 객체, 변수 등 어떤 유형의 값이든 될 수 있음.
const person = {
   name: "John",
   age: 30,
};

module.exports = person;

person 객체를 module.exports에 할당하여 외부로 내보내고 있음 / 이를 통해 다른 모듈에서 해당 객체를 가져와 사용할 수 있음

모듈의 내보내기를 통해 현재 모듈에서 외부로 내보낼 함수, 객체, 변수 등을 정의할 수 있음
코드의 재사용성을 높이고, 모듈 간의 상호작용을 구현 가능

5) 모듈의 재사용
한 번 정의한 모듈을 다른 파일에서 가져와서 사용하는 것을 의미
코드의 중복을 피하고, 모듈화된 기능을 여러 곳에서 활용

모듈의 재사용 방법
1. require() 함수를 사용하여 다른 파일에서 모듈을 가져옴
const greet = require("./greet");

greet("AI");

greet 모듈을 다른 파일에서 가져와 사용 / require() 함수를 사용하여 가져온 모듈은 해당 모듈에서 정의한 함수, 객체, 변수 등을 활용

2. 가져온 모듈의 기능을 활용하여 원하는 기능을 수행
const calculator = require("./calculator");

const result = calculator.add(5, 10);
console.log(result); // 15

calculator 모듈을 가져와 add() 함수를 호출하여 두 수의 합을 계산 / 가져온 모듈의 함수를 활용하여 필요한 기능을 구현

한 번 작성한 모듈을 여러 파일에서 가져와 사용
코드의 중복을 피하고, 모듈화된 기능을 효율적으로 재사용
모듈 간의 상호작용을 구현하여 코드의 유지보수성을 향상
