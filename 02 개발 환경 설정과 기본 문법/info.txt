개발 환경 설정과 기본 문법
- 개발을 시작하기 위해 필요한 환경 구성과 기본적인 JavaScript 문법을 다루는 부분
- 설치 및 환경 구성 : 설치하는 방법과 개발 환경을 구성하는 방법 / 설치 과정과 환경 변수 설정 등을 진행하여 개발 환경을 구축하는 방법을 안내
- 기본적인 JavaScript 문법 : JavaScript는 Node.js 개발에 필수적인/기본적인 언어 / 변수, 데이터 타입, 연산자, 조건문, 반복문 등 JavaScript의 기본 문법에 대해 설명
- CommonJS 모듈 시스템 : CommonJS 모듈 시스템을 사용하여 모듈을 정의하고 재사용 / CommonJS 모듈 시스템의 사용 방법과 모듈을 생성하고 불러오는 방법 / 모듈의 내보내기와 가져오기 방식
