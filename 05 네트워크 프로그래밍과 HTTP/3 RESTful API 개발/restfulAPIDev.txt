RESTful API 개발
Node.js를 사용하여 RESTful API를 개발하는 방법 소개
RESTful API는 웹 애플리케이션과 클라이언트 간의 상호작용을 위한 표준화된 방법론으로, HTTP 프로토콜을 기반으로 데이터를 요청하고 응답하는 API

RESTful API의 개념과 REST 아키텍처 스타일에 대해 소개
REST는 Representational State Transfer의 약자로, 자원을 URI로 표현하고 HTTP 메서드를 사용하여 해당 자원에 대한 CRUD(Create, Read, Update, Delete) 작업을 수행하는 방식
클라이언트는 간편한 방법으로 서버의 데이터를 조작

Node.js를 사용하여 RESTful API를 개발하는 방법 소개
Express.js와 같은 Node.js 프레임워크를 활용하여 API 엔드포인트를 정의
각 엔드포인트에 대한 요청을 처리하는 핸들러 함수를 작성
핸들러 함수를 통해 데이터베이스와의 상호작용을 수행하고, 클라이언트에게 적절한 응답을 반환

API의 버전 관리, 인증과 권한 부여, 오류 처리 등의 추가적인 기능에 대해서도 소개
RESTful API를 개발할 때는 이러한 기능들을 고려하여 안정적이고 보안성 있는 API를 구축

Postman과 같은 API 도구를 사용하여 개발한 API를 테스트하고 디버깅하는 방법에 대해서도 소개
API 도구를 사용하여 API의 요청과 응답을 확인하고, 필요한 경우 수정 및 개선 작업을 수행

Node.js와 Express.js를 사용하여 RESTful API를 개발하는 예제 코드

const express = require('express');
const app = express();
const port = 3000;

// 데이터베이스 대신 간단한 배열을 사용하여 예시 데이터 생성
const users = [
  { id: 1, name: 'John Doe' },
  { id: 2, name: 'Jane Smith' },
];

// 모든 사용자 정보 반환
app.get('/users', (req, res) => {
  res.json(users);
});

// 특정 사용자 정보 반환
app.get('/users/:id', (req, res) => {
  const userId = parseInt(req.params.id);
  const user = users.find(user => user.id === userId);

  if (user) {
    res.json(user);
  } else {
    res.status(404).json({ error: 'User not found' });
  }
});

// 새로운 사용자 생성
app.post('/users', (req, res) => {
  const newUser = { id: users.length + 1, name: req.body.name };
  users.push(newUser);
  res.status(201).json(newUser);
});

// 사용자 정보 업데이트
app.put('/users/:id', (req, res) => {
  const userId = parseInt(req.params.id);
  const user = users.find(user => user.id === userId);

  if (user) {
    user.name = req.body.name;
    res.json(user);
  } else {
    res.status(404).json({ error: 'User not found' });
  }
});

// 사용자 삭제
app.delete('/users/:id', (req, res) => {
  const userId = parseInt(req.params.id);
  const userIndex = users.findIndex(user => user.id === userId);

  if (userIndex !== -1) {
    users.splice(userIndex, 1);
    res.sendStatus(204);
  } else {
    res.status(404).json({ error: 'User not found' });
  }
});

// 서버 실행
app.listen(port, () => {
  console.log(`Server is running on http://localhost:${port}`);
});

Express.js를 사용하여 간단한 사용자 정보를 다루는 RESTful API를 구현한 예제
/users 엔드포인트는 모든 사용자 정보를 반환
/users/:id 엔드포인트는 특정 사용자의 정보를 반환
/users에 POST 요청을 보내면 새로운 사용자를 생성
/users/:id에 PUT 요청을 보내면 해당 사용자의 정보를 업데이트
/users/:id에 DELETE 요청을 보내면 해당 사용자를 삭제

코드를 실행하면 Express.js 서버가 3000번 포트에서 실행되며, API 엔드포인트에 대한 요청을 처리

코드를 참고하여 RESTful API를 개발하면, 클라이언트와 서버 간의 데이터 교환을 표준화된 방식으로 처리
API의 엔드포인트와 HTTP 메서드를 적절하게 정의하고, 요청에 따라 적절한 응답을 반환하는 방식으로 API를 설계
