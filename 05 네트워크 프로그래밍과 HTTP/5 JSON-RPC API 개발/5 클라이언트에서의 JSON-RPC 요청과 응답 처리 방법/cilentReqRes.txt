클라이언트에서의 JSON-RPC 요청과 응답 처리 방법
cURL을 사용한 JSON-RPC 요청 예제
curl -X POST -H "Content-Type: application/json" -d '{
  "jsonrpc": "2.0",
  "method": "add",
  "params": [2, 3],
  "id": 1
}' https://example.com/rpc-endpoint

cURL을 사용하여 JSON-RPC 요청
-X POST는 POST 메서드를 사용
-H "Content-Type: application/json"는 요청 헤더에 Content-Type을 JSON으로 설정
-d 옵션을 사용하여 요청 본문에 JSON-RPC 요청 메시지를 포함

jQuery를 사용한 JSON-RPC 요청 예제
$.ajax({
  url: 'https://example.com/rpc-endpoint',
  type: 'POST',
  contentType: 'application/json',
  data: JSON.stringify({
    jsonrpc: '2.0',
    method: 'add',
    params: [2, 3],
    id: 1
  }),
  success: function(response) {
    // 요청 성공 시 처리 로직
    console.log(response.result);
  },
  error: function(error) {
    // 요청 실패 시 처리 로직
    console.error(error);
  }
});

jQuery를 사용하여 JSON-RPC 요청을 보내는 방법
$.ajax() 함수를 사용하여 요청을 보내고, success 콜백 함수를 통해 응답을 처리
