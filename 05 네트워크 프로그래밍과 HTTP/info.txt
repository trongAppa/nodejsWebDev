네트워크 프로그래밍과 HTTP
- HTTP 프로토콜에 대한 이해를 바탕으로 서버 구축 및 API 개발
- HTTP 프로토콜 이해에서는 HTTP 프로토콜의 기본 개념과 동작 방식, 요청과 응답의 구조 등 공부 / 웹 애플리케이션 개발에 필요한 기본 지식
- HTTP 모듈을 사용한 서버 구축에서는 Node.js의 내장 모듈인 http 모듈을 활용하여 간단한 웹 서버를 구축하는 방법을 학습 /서버의 생성, 요청 처리, 응답 전송 등의 과정을 다루며, 실제로 동작하는 웹 서버를 구현
- RESTful API 개발에서는 RESTful 아키텍처 스타일을 따르는 API 개발 / REST의 개념과 원칙, 자원과 URI 설계, HTTP 메서드 활용 등을 학습하고, Express.js와 같은 프레임워크를 사용하여 RESTful API를 개발하는 방법을 실습
- GraphQL에서는 GraphQL 쿼리 언어와 서버를 사용하여 데이터를 효율적으로 요청하고 응답하는 방법을 학습 / GraphQL의 기본 개념, 스키마 정의, 쿼리 및 뮤테이션 작성 방법 등을 다루며, Apollo Server와 같은 도구를 사용하여 GraphQL API를 개발하는 방법을 실습
- JSON-RPC에서는 JSON-RPC 프로토콜을 활용하여 원격 프로시저 호출(RPC)을 구현하는 방법을 학습 / JSON-RPC의 개념, 요청 및 응답 형식, 클라이언트와 서버 간의 통신 방법 등을 다루며, JSON-RPC를 사용하여 분산 시스템이나 마이크로서비스 아키텍처에서의 통신을 구현하는 방법을 실습
- REST API, GraphQL, JSON-RPC 비교에서는 REST API, GraphQL, JSON-RPC의 특징, 장단점, 사용 시나리오 등을 비교 분석 / 각 프로토콜의 특성과 적합한 사용 사례에 대해 알아보며, 언제 어떤 프로토콜을 선택해야 하는지에 대한 판단 기준을 익힘
