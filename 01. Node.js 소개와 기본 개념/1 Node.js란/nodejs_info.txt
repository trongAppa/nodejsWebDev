Node.js란? 
- 구글 크롬의 V8 JavaScript 엔진으로 빌드된 JavaScript 런타임 환경
- 서버 사이드에서 JavaScript를 실행할 수 있는 환경을 제공
- 비동기 프로그래밍과 이벤트 기반 모델을 기반으로 동작하여 많은 요청을 동시에 처리하고, 대용량의 데이터를 효율적으로 처리가능
- 단일 스레드 이벤트 루프를 사용하여 비동기식 I/O 작업을 처리
- 논블로킹(non-blocking) 방식으로 동작하며, I/O 작업이 완료될 때까지 기다리지 않고 다른 작업을 처리 가능
- 다중 요청을 동시에 처리하고, 많은 클라이언트를 동시에 처리하는 데 용이
- JavaScript 언어를 사용하여 프론트엔드와 백엔드를 동일한 언어로 개발할 수 있어 개발 생산성을 높임
- 모듈 생태계가 풍부하여 다양한 모듈을 사용하여 개발 속도를 향상
- 높은 처리량과 확장성을 제공하는 동시에 개발자들에게 편리한 개발 환경을 제공
- 웹 애플리케이션 개발뿐만 아니라 네트워크 프로그래밍, 서버 사이드 스크립팅, API 서버 등 다양한 영역에서 활용
- 현대적이고 효율적인 웹 개발을 위한 강력한 도구로 인정 받음

JavaScript를 사용하여 프론트엔드와 백엔드를 동일한 언어로 개발할 수 있어 개발자들의 생산성을 높임
비동기 프로그래밍과 이벤트 기반 모델을 통해 많은 요청을 동시에 처리하고, 높은 처리량과 확장성을 제공
