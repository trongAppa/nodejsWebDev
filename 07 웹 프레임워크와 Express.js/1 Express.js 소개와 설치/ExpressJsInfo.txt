Express.js 소개와 설치
Express.js 소개
- Node.js를 기반으로 한 경량화면 프레임워크로, 웹 애플리케이션을 빠르고 간편하게 개발할 수 있도록 도와줌 
- 간단하고 직관적인 API를 제공하여 개발자가 웹 애플리케이션의 동작을 더욱 세밀하게 제어할 수 있음
- 다양한 기능과 라이브러리를 활용하여 웹 애플리케이션을 구축할 수 있습니다.

Express.js 설치
- npm install express

실행하면 Express.js와 그 종속성들이 프로젝트에 설치
설치가 완료되면 Express.js를 사용하여 웹 애플리케이션을 개발할 준비

Node.js의 강력한 기능을 활용하여 웹 애플리케이션을 구축하는데 사용되는 프레임워크
간단하고 직관적인 API를 제공하여 개발자가 웹 애플리케이션의 동작을 제어할 수 있으며, 다양한 기능과 라이브러리를 활용하여 효율적인 웹 애플리케이션을 개발
