Node.js
- 구글 크롬의 V8 JavaScript 엔진으로 빌드된 JavaScript 런타임 환경
- 서버 사이드에서 JavaScript를 실행할 수 있는 환경을 제공하며, 비동기 프로그래밍과 이벤트 기반 모델을 기반으로 동작

비동기 프로그래밍과 이벤트 기반 모델
- 단일 스레드 이벤트 루프를 사용하여 비동기식 I/O 작업을 처리하며, 논블로킹(non-blocking) 방식으로 동작
- 많은 요청을 동시에 처리하고, 대용량의 데이터를 효율적으로 처리

Node.js의 장점과 특징
- 높은 처리량과 확장성을 제공하며, JavaScript를 사용하여 프론트엔드와 백엔드를 동일한 언어로 개발할 수 있어 개발 생산성이 높음
- Node.js는 모듈 생태계가 풍부하여 다양한 모듈을 사용하여 개발 속도가 높음
